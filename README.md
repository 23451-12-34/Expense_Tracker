## 🚀 Features

✅ Add new expenses with description  
✅ Filter expenses by **date** and **category**  
✅ View total & category-wise expense summary  
✅ 📊 Visual monthly and annual reports (Chart.js)  
✅ 🗑️ Delete any expense
